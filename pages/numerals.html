Summary

Elves use the duodecimal system, which is a base-12 system. Sindarin has a decimal system, but the duodecimal can be extrapolated.

The numerals are written in crescent order of magnitude, so it starts at a unit, then decimal unit, hundreds, thousands, etc. Example: 

  Decimal ..: 74035
  English ..: Seventy four thousand thirty five
  Sindarin .: Leben nelphaen meneg canad odophaen
