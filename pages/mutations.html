Summary

Many words in Sindarin change sounds when spoken in a certain sequence.

These changes are known as consonant mutations. These apply to the initial consonant of a word.

Sindarin has about six types of consonant mutations: soft, nasal, mixed, stop, liquid, and sibilant. Examples: 

  Soft:
  bâr    = "house" → vâr
  
  Nasal:
  bâr    = "house" → mâr
  
  Mixed:
  bâr    = "house" → mbar
  
  Stop:
  galad  = "light" → ñgalad
  
  Liquid:
  galadh = "tree"  → 'aladh'
  
  Sibilant:
  loss   = "snow"  → lhoss
